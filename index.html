Nature Emploi

Nature Emploi est une application mobile android permettant de collecter des differentes espèces de plantes. Nature Emploi est un projet personnel d'application qui m'a permis de prendre en main une certaine technologie tels que Android Studio, Firebase et Kotlin

AH-Fashion

AH-Fashion est une application des mise en place place pour faciliter la tâche à tous les couturiers et toutes personnes souhqitants confectionner des vêtements depuis son lieu d'emplacement !

MoMo-Foods

MoMo-Foods est une application Web responsive permettant de faire la commande de restauration des plats Africains aussi bien que les plats Europeen. Cette application web a été realisé avec les technologies suivantes : PHP pour faire le Backend, MySQL pour la gestion de la Base de donnée et GitHub pour gérer les versions ...
